# EduPlan Namibia - Teacher Survey                             
                                                                
## About This Survey                                            
This anonymous questionnaire gathers feedback from Namibian teachers about their administrative challenges and needs.Purpose:** Understand teacher pain points to build effective solutions for reducing administrative workload.Privacy:** 100% anonymous - only regional data collected Time:** 8-10 minutes to complete           ## Survey Link  [Complete the Survey](https://kabendestorm.github.io/eduplan-teacher-survey-/) 
## About EduPlan                                           
EduPlan is a digital platform designed to automate teacher administrative tasks including:             
- Lesson planning                                     
- Schemes of work                                               
- CA mark calculations                                          
- Report card generation                                        
- Performance tracking
- Built by teachers, for teachers in Namibia.
